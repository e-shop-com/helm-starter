Helm-starter
===

A helm starter to create a chart for [ethibox](https://github.com/ston3o/ethibox).

## Usage

```bash
git clone https://github.com/ston3o/helm-starter ~/.helm/starters/
helm create -p helm-starter <chart_name>
```
